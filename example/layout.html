Layout

Built on a responsive fixed and fluid 16-column grid. Hybrid 1140px grid system.

Layout Basics

Fixed & Fluid Grid Layout

The default grid system utilizes 16 columns grid, with max-width for the fluid grid sets to 1140px (includes 20px padding wrapper).

In order to use the fixed grid layout, please use: <div class="row">…</div>. While the fluid grid layout can be deployed by using <div class="row-fluid">.

Responsive Design

The default breakpoints for responsive design are:

  • max-width: 320px. For devices with maximum width of 320px (portrait orientation in iPhone).
  • max-width: 480px. For devices with maximum width of 480px (landscape orientation in iPhone).
  • max-width: 768px. For devices with maximum width of 768px (e.g. iPads orientation portrait and other tablets).
  • max-width: 1024px. For devices with maximum width of 1024px (e.g. iPads, tablets orientation landscape and netbooks).
