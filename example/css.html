Typography (header, paragraphs, lists and other type elements)

Typography base settings

The typographic rhythm based on @baseFontSize and @baseLineHeight.

  • @baseFontSize, @baseLineHeight and @baseFontSizeinPX to define the base font size and line-height.
  • @baseFontColor defines the base font color.
  • @linkColor, @linkColorHover, @linkColorVisited define states for text with links.
  • @baseSelectionColor defines the color for selection text/part.

Button Type

Button Class Name Description
.button Default button style, standard with gradient.
.button .btn-neutral Neutral style, used as an alternate to default style
.button .btn-primary Primary button style, used for default/primary actions
.button .btn-warning Indicates for action that needs to be taken cautiously
.button .btn-error Button style for an action that could impact a dangerous or negative state

Table options

Name/Type Class Description
Default None No styles, just bareboned default table.
Basic .table No vertical border/line, only lines between rows.
Bordered .table-bordered Outer border and rounded corner.
Striped .table-striped Odd/even rows.
Condensed .table-condensed Cuts vertical padding to 4px and horizontal padding to 5px.
