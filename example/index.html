What is Frontkit?

FrontKit is a custom tailored, less bloated web front-end development kit created to assist you to develop the front-end aspects of your web development projects.

FrontKit's current key features are:

  • Using the latest HML5 and CSS3 properties
  • Serving both barebone (the minimum needs for your front-end development) and fully customizable example version with 16-column grid.
  • Responsive design. Components are scaled according to a range of resolutions and devices.
  • Built on LESS, with fully customizable variables, nesting, operations and mixins to your needs for efficient coding.

Requirements and Acknowledgements

FrontKit's HTML and CSS properties requires HTML5 doctype, but despite all that FrontKit also inspired from and even makes use of some of the following libraries and resources:

  • jQuery
  • LESS
  • Modernizr
  • HTML5 Boilerplate
  • Twitter Bootstrap
